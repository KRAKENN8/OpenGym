# OpenGym
OpenGym on avatud lähtekoodiga veebi- ja mobiiliplatvorm oma jõusaali, spordiklubi või treeningruumi loomiseks ja haldamiseks. Projekt pakub täielikku tööriistade ja dokumentatsiooni komplekti süsteemi iseseisvaks juurutamiseks, kohandamiseks ja edasiarendamiseks vastavalt teie vajadustele.
